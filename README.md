## 项目介绍
  react项目脚手架
  
## 如何使用
  
  npm install
  `环境依赖node-sass，保证当前环境有node-sass支持`
  npm start 启动项目
  
  npm run dist 发布项目
  
  
## 脚手架目录
    cfg——存放环境运行的配置文件
    dis——编译后的文件夹
    src——开发源代码
      |——components（组件）
      |——public
      |    |--font（字体文件夹）
      |    |--imgs（图片文件夹）
      |    |--lib （第三方JS库文件夹）
      |    |--style（公共样式文件夹）
      |——sources（测试用的数据）
      |——favicon.ico 
      |——index.html （主页面）
      |——index.js （入口文件）
